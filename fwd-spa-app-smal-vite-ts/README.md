The Simple SPA application to be deployed on AWS as a static page via CloudFront.
AWS Configuration icnludes:
1. Custom Domain.
2. Custom Certificate.
3. CloudFront distribution.

Url to website:
[react-spa.smal.click](https://react-spa.smal.click/)

## Setup
1. To run locally: ```npm install```
2. To build: ```npm installreact-router-dom```
   
## Usage
1. To run locally: ```npm run dev```
2. To build: ```npm run build```
